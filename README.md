# Gerador de Etiquetas e Romaneio - Ranor

Sistema simples desenvolvido em **HTML, CSS e JavaScript** para automatizar processos de envio de equipamentos na empresa Ranor.

## Objetivo

O sistema foi criado para **otimizar tarefas operacionais do dia a dia**, reduzindo trabalho manual e padronizando o processo de envio de equipamentos.

## Funcionalidades

- Geração automática de **etiquetas de envio**
- Geração de **romaneio de equipamentos**
- **Envio automático de e-mail ao financeiro** solicitando emissão de nota fiscal de envio
- Interface simples e rápida para uso interno

## Tecnologias

- HTML
- CSS
- JavaScript

## Motivação

Projeto criado para **automatizar processos repetitivos do trabalho**, aumentando produtividade e reduzindo erros operacionais no fluxo de envios.
